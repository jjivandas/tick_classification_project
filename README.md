# Tick Classification with BioCLIP

This project implements a full pipeline for tick species classification using [BioCLIP](https://imageomics.github.io/bioclip/) embeddings and SVM few-shot classification. It covers data cleaning, zero-shot inference, few-shot SVM experiments with Monte Carlo evaluation, and SVM hyperparameter optimization.

## Notebook Details

### Notebook 01 — Data Exploration & Cleaning

Cleans raw CSV data and produces the classification dataset.

**Steps:**
- Loads metadata CSV, deduplicates by Sample ID
- Cross-references CSV entries with image files
- Normalizes species labels (fixes typos)
- Removes generic "Ixodes" (no species-level ID)
- Removes Ixodes cookei duplicates (Sample IDs ending in "a")
- Removes broken specimens (identified via error analysis)
- Annotates pathogen results and tick condition
- Filters to species with >= 50 specimens
- Saves species distribution bar chart

**Outputs:** `final_data_cleaned.json` (1442 images, 721 specimens, 5 species)

### Notebook 02 — BioCLIP Zero-Shot Inference

Runs BioCLIP zero-shot classification on all images as a baseline.

**Outputs:** `results/bioclip_zeroshot/runs/{timestamp}/bioclip_results.npz`

### Notebook 03 — Few-Shot SVM (Default Parameters)

Trains SVM classifiers on BioCLIP embeddings with default hyperparameters (C=1, gamma=scale, rbf).

**Method:** 100 Monte Carlo train/test splits at each shot level (1, 3, 5, 10, 25, 35 shots per species).

**Outputs:** `results/svm/runs/{timestamp}/` with predictions, per-species accuracy, confusion matrices, macro accuracy curves, and combined per-species + macro accuracy overlay plot.

### Notebook 04 — SVM Hyperparameter Optimization

Finds optimal SVM hyperparameters via GridSearchCV, then re-runs the full Monte Carlo sweep.

**Method:**
1. GridSearchCV (5-fold stratified CV) across 20 MC splits at K=35
2. Select most frequent best (C, gamma, kernel) combination
3. Full 100-run sweep with optimized params across all shot levels

**Outputs:** `results/svm_hp/runs/{timestamp}/` with:
- Hyperparameter search results
- Default vs optimized comparison plots
- Per-species + macro accuracy overlay plots (for both default and optimized)
- Row-normalized confusion matrices at each K
- Side-by-side default vs optimized confusion comparison

**Filtering applied:**
- Generic "Ixodes" genus-only labels removed
- Ixodes cookei duplicate Sample IDs removed
- 17 broken specimens removed (data quality issues)
- Species with < 50 specimens excluded

## Key Design Decisions

- **Dorsal + ventral averaging:** Each specimen is represented by the mean of its two BioCLIP embeddings
- **Embedding cache:** Embeddings cached as `.npy` files keyed by `sha256(image_path)[:24]`
- **Deterministic seeding:** `seed_for(k, run_id)` ensures reproducible train/test splits
- **Monte Carlo evaluation:** 100 random splits per shot level for stable accuracy estimates with 95% CIs
- **Separate results directories:** Default SVM (`results/svm/`) and HP-optimized (`results/svm_hp/`) never overwrite each other; each run gets a timestamped folder
